## Prerequisites

Download the *Diabetes* dataset from üëâ [Diabetes Dataset](https://www.kaggle.com/datasets/iammustafatz/diabetes-prediction-dataset) and extract the folder and paste the csv file in the root directory.



## Deployment

To run this project first clone this repository using:

```bash
  https://github.com/aka-Harsh/Diabetes-Predictor.git
```
Locate this repository using terminal and then create a virtual enviroment and activate it:

```bash
  python -m venv venv
  .\venv\Scripts\activate
```
Perform this in your VScode editor to select python intepreter:
```bash
  Select View > Command Palette > Python: Select Interpreter > Enter Interpreter path > venv > Script > python.exe
```

Install all the required packages:
```bash
  pip install -r requirements.txt
```

Finally run the app.py file:
```bash
  python app.py
```
